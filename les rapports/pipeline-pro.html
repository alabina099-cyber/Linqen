04 Continuous Deployment (CD) Deploy only after full CI + Security green
Docker Image Build → GHCR (GitHub Container Registry)
4-stage multi-stage Dockerfile: base → deps → builder → runner (production only)
Base image: node:22-alpine — minimal attack surface (~150 MB final image)
Non-root user: UID 1001 — hardened container, no root privileges at runtime
GHA Docker layer cache (cache-from/to: type=gha) — ~65% rebuild time reduction
3 tags per build: latest · main · sha-<commit> — full traceability via sha256 digest
