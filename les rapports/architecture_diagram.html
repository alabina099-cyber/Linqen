Qlinqen — Hybrid Multi-Component Architecture
Chrome Extension + Web Dashboard + Headless Worker + Backend Services
Chrome Extension (Manifest V3)
Background Service Worker
Polls action queue, enforces daily limits, applies human-like delays, manages authentication, and recovers stale actions.
Content Script
Scrapes profiles and search results, performs automated actions (connections, messages), and monitors inbox replies directly on LinkedIn pages.
Bridge Script
Relays commands and authentication tokens between the web dashboard and the extension via the browser.
Web Dashboard
Next.js + React + TypeScript
Full-stack application with server-side rendering, API routes for business logic, middleware for security, and a modern component-based UI.
Core Modules
Campaign creation and targeting, AI message approval queue, agent chat interface, automation settings, and real-time analytics dashboard.
Authentication
LinkedIn cookie-based login for admins, email and password with JWT for team members, and role-based access control.
Headless Worker
Node.js + Puppeteer
Headless browser automation that polls the same action queue as the extension, enabling 24/7 server-side execution without a browser.
Containerized Deployment
Packaged as a Docker container, deployed independently via Coolify on Oracle Cloud for scalable, always-on automation.
Extension → HTTPS / REST + JWT ← Worker
Backend & Infrastructure
PostgreSQL
Cloud-hosted relational database for structured data storage including prospects, campaigns, action queues, and audit logs.
AI / LangChain / OpenAI
Intelligent message generation and campaign execution using large language models with configurable tone and style.
Nginx Load Balancer
Reverse proxy with TLS termination, rate limiting, and static asset caching to handle incoming traffic securely.
