Chrome Extension (Manifest V3)
Background Service Worker
Polls action queue (12 s), enforces daily limits, applies human-like delays, monitors inbox replies & accepted connections, recovers stale actions.
Content Script
Scrapes LinkedIn profiles and search results, performs DOM-level automated actions with selector fallbacks.
Bridge Script
Relays commands and JWT tokens between the web dashboard and the extension.
Web Dashboard
Next.js 16 · React 19 · TypeScript
Full-stack app with server-side rendering, API routes, middleware, and a component-based UI.
Core Modules
Campaigns, approval queue, agent chat, automation settings, and real-time analytics dashboard.
Authentication
LinkedIn cookie login for admins · Email/password + JWT for team members · Role-based access control.
Headless Worker
Node.js + Puppeteer
Polls the action queue every 5 s and executes LinkedIn actions server-side 24/7 without a browser extension.
Containerized Deployment
Packaged as a Docker container, deployed via Coolify on Oracle Cloud. Supports horizontal scaling.
Extension ⟶ HTTPS / REST + JWT ⟵ Worker
Backend & Infrastructure
PostgreSQL (Neon)
Serverless relational database for prospects, campaigns, action queues, and audit logs.
AI · LangChain · OpenAI
Intelligent message generation and campaign execution via tool-calling agents with configurable tone.
Nginx Load Balancer
Reverse proxy with TLS termination, rate limiting, and static asset caching. Horizontal scaling via Docker Compose.
Observability Stack
Prometheus · Grafana · Loki · AlertManager · cAdvisor · Uptime Kuma — metrics, logs, alerts, and uptime monitoring.
